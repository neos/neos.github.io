IfAuthenticatedViewHelper

class IfAuthenticatedViewHelper extends AbstractConditionViewHelper (View source)

This view helper implements an ifAuthenticated/else condition.

= Examples =

This is being shown whenever a user is logged in ``` Everything inside the tag is being displayed if you are authenticated with any account. This is being shown in case you have access. This is being displayed in case you do not have access. ``` Everything inside the "then" tag is displayed if you have access. Otherwise, everything inside the "else"-tag is displayed.

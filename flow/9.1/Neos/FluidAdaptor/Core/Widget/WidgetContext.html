WidgetContext

class WidgetContext (View source)

The WidgetContext stores all information a widget needs to know about the environment.

The WidgetContext can be fetched from the current request as internal argument __widgetContext, and is thus available throughout the whole sub-request of the widget. It is used internally by various ViewHelpers (like , , ), to get knowledge over the current widget's configuration.

It is a purely internal class which should not be used outside of Fluid.

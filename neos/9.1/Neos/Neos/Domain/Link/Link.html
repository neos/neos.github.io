Link

final class Link implements JsonSerializable (View source)

Link value object modeled partially after the html spec for tags.

Note that currently the link editor can only handle and write to the property {\Neos\Neos\Domain\Link\Link::$target} "_blank" | null and to {\Neos\Neos\Domain\Link\Link::$rel} ["noopener"] | null

The Link values can be accessed in Fusion the following:

href = ${q(node).property("link").href}
title = ${q(node).property("link").title}
# ...

In case you need to cast the uri in {\Neos\Neos\Domain\Link\Link::$href} explicitly to a string you can use: String.toString(link.href)

DebugConsoleImplementation

class DebugConsoleImplementation extends AbstractArrayFusionObject (View source)

A Fusion object for debugging fusion-values via the browser console

//fusionPath value The variable to serialize and output to the console. //fusionPath title Optional custom title for the debug output. //fusionPath method Optional alternative method to call on the browser console. //fusionPath content When used as process the console script will be appended to it.
